# 9router.id

Landing page untuk komunitas WhatsApp 9router.id: developer, AI, vibe coding, konfigurasi 9 router, jaringan, dan teknologi praktis di Indonesia.

## Stack

- Astro
- Tailwind CSS
- TypeScript
- Cloudflare Pages

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output build: `dist/`.
